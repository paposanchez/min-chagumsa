## ZLARA CMS FRAMWORK
Laravel + Bootstrap + Components 기반의 CMS 템플릿 프레임워크

### Installation
1. download ZLARA source and make new project
2. npm install 
3. composer install
4. modify .env
5. run "php artisan migrate"
6. run "php artisan db:seed"
7. run "php artisan serve"
